# Invitur 🎉

**Invitaciones digitales para todo tipo de evento**

Plataforma de invitaciones virtuales animadas para bodas, XV años, cumpleaños, bautizos, graduaciones, baby shower, aniversarios y eventos corporativos.

## 🎨 Características

- 8 tipos de evento con plantillas únicas
- 9 colecciones de diseño filtrables
- Modo claro / oscuro
- Formulario de contacto con simulación de envío
- Diseño responsive (mobile-first)
- Animaciones CSS fluidas por tipo de evento

## 📁 Estructura

```
invitur/
├── index.html          ← Portfolio principal

```
